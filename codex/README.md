# Codex GLM52 Files

This directory contains the Codex-specific GLM52 profile and launcher scripts.

- `profiles/glm52.config.toml`: profile-scoped GLM model/provider settings
- `scripts/start-proxy.*`: start the local Responses-to-Chat proxy
- `scripts/stop-proxy.*`: stop the local proxy
- `scripts/start-codex-glm52.*`: start the proxy and launch Codex with
  `--profile glm52`
- `SESSION_AND_MODEL_NOTES.md`: operational notes for `/model`, `/resume`, and
  provider-scoped session behavior
